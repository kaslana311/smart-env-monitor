# 智能环境监测系统 (Smart Environment Monitor)

基于 STM32F407IG + RT-Thread 的嵌入式环境监测项目。

## 项目简介

本项目实现了一个完整的嵌入式环境监测系统，采用 RT-Thread 实时操作系统进行多线程任务调度，
使用多种 IPC 机制实现线程间通信与同步。

## 硬件平台

- MCU: STM32F407IG (Cortex-M4)
- 操作系统: RT-Thread v4.0.2
- 开发环境: Keil MDK / RT-Thread Studio

## 系统架构

### 线程设计 (4个独立线程)

| 线程 | 优先级 | 周期 | 功能 |
|------|--------|------|------|
| sensor | 15 | 500ms | 传感器数据采集、心跳LED控制 |
| process | 16 | 事件驱动 | 数据滤波处理、阈值检测 |
| display | 17 | 事件驱动 | 串口格式化输出 |
| alarm | 18 | 200ms | 告警监控、LED闪烁指示 |

### IPC 通信机制

- **消息队列**: sensor → process (传感器数据传输)
- **互斥量**: 保护系统全局状态变量
- **信号量**: process → display (数据就绪通知)

## 功能特性

- 模拟温度(-10~45°C)、湿度(20~95%)、光照(0~1000 lux)采集
- 三点中值滤波 + 移动平均滤波 双重滤波
- 数据质量评估算法
- 高温/高湿/低光照 三级告警
- LED 多种闪烁模式（快闪/慢闪/双闪/常亮）
- 告警迟滞防抖，避免边界振荡
- 简洁/详细双显示模式
- 数据变化趋势指示

## Git 协作流程

1. 各成员在独立功能分支开发 (driver-main / thread-function / display-module)
2. 开发完成后创建 Pull Request
3. 由其他成员交叉审核代码
4. 审核通过后合并至 main 分支
5. 出现冲突时协商解决，禁止强制覆盖
